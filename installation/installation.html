📋 Venge.io Mod Installation Guide

Follow the steps below to properly install and use the mod on your computer.

🖥️ The mod can only be used on Venge Client:

Official Venge Client Download Link: https://social.venge.io/client.html

📁 Folder Structure:

CharmMod
│
├── ReadMe.txt         ← This installation guide  
└── Venge Client       ← The main mod folder

📥 Installation Steps:

  1. Move the entire Venge Client folder to your Documents directory.
  2. Example path: C:\Users\YourName\Documents\Venge Client
  3. Ensure the folder name remains exactly as "Venge Client" for proper functioning.
